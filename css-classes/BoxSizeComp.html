Box-Sizing Comparison

box-sizing: content-box; (Default)
box-sizing: border-box;

How it works:

Both boxes are set to width: 300px, padding: 20px, and border: 10px.

  • The content-box (left) adds padding and border *outside* its 300px width.
    Total Width = 300px (width) + 40px (padding) + 20px (border) = 360px.
  • The border-box (right) includes padding and border *inside* its 300px width.
    Total Width = 300px.

border-box is generally more intuitive for layout design.
