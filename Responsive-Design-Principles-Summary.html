Fluid Layouts

Layouts that adapt to available width using relative units like % and fr.

grid-template-columns: 1fr 2fr;

Flexible Media

Use max-width:100% and object-fit for images and videos.

img{ max-width:100%; height:auto; }

Breakpoints

Choose breakpoints for common device widths and adjust layout.

@media (max-width:600px){...}
